# Mandalay Travel Web Project

## Description
Mandalay Travel is a JSP-based web application developed to provide information about Mandalayâ€™s tourist attractions, culture, food, festivals, and music. The project aims to promote digital tourism while enhancing academic knowledge of web application development.

## Technologies Used
- HTML, CSS, JavaScript
- JSP & Servlet
- MySQL
- Apache Tomcat

## Features
- Interactive navigation system
- Image gallery with Previous and Next functionality
- Sections for Food, Festivals, Places, and Music
- Responsive and user-friendly interface

## How to Run the Project
1. Install Apache Tomcat
2. Import the project into Eclipse or VS Code
3. Configure MySQL database
4. Deploy the project on Tomcat server
5. Access via browser
